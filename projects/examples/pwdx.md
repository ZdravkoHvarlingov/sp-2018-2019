# `pwdx`

## Примери за употреба

### Пример 1.

`$$` е променлива на shell-а, в която се съхранява PID на текущия процес (т.е. в случая на shell-а).

```
$ pwdx $$
12494: /home/sid
```

### Пример 2.

```
$ ps
  PID TTY          TIME CMD
12494 pts/0    00:00:00 bash
13757 pts/0    00:00:00 ps
18969 pts/0    00:00:06 zsh

$ pwdx 12494 18969
12494: /home/sid
18969: /home/sid/Documents
```
